Déposez vos images ici !

C'est le **meilleur endroit** pour stocker les images que vous allez lier dans le fichier `src/content.ts`.

**Comment faire ?**
1. Glissez-déposez vos photos (ex: `portrait.jpg`, `cabinet.png`) dans ce dossier `public/images/` via l'explorateur de fichiers à gauche.
2. Ouvrez le fichier `src/content.ts`.
3. Remplacez les liens existants (qui commencent par `https://...`) par le chemin de votre image, en commençant par `/images/`.

**Exemple :**
Si vous avez ajouté `mon-cabinet.jpg` ici, modifiez `content.ts` comme ceci :
`image: "/images/mon-cabinet.jpg",`
